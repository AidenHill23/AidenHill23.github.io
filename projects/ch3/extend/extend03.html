Modifying Images

It is very important to use descriptive file names for image files so you will never have to worry about mixing them up. For example you could have 2 images, one has a man holding a basketball while the other one has a women holding a basketball. Imagine if you named the files basketball1 and basketball2. It would be hard to tell which one is different it would be better if you named them more discriptively.
